# Marketing Funnel & Conversion Performance Analysis

## 📌 Project Overview
This Power BI dashboard analyzes marketing funnel performance and customer conversion behavior using the Bank Marketing Dataset. The objective is to identify key factors influencing conversions and provide insights that help improve marketing campaign effectiveness.

---

## 🎯 Objectives
- Analyze customer conversion performance.
- Identify high-converting customer segments.
- Evaluate the effectiveness of different contact methods.
- Understand conversion trends across months and demographics.
- Provide actionable recommendations to improve conversion rates.

---

## 📊 Key Performance Indicators (KPIs)

### Total Contacts
Total number of customers contacted during marketing campaigns.

### Converted Customers
Number of customers who successfully subscribed after being contacted.

### Conversion Rate (%)
Measures the percentage of contacted customers who converted.

**Formula:**

```DAX
Conversion Rate =
DIVIDE([Converted Customers], [Total Contacts], 0) * 100
```

---

## 📈 Dashboard Visualizations

### 1. Contact Method vs Converted Customers
Analyzes conversion performance across different communication channels.

### 2. Monthly Conversion Analysis
Displays conversion trends by month to identify peak-performing periods.

### 3. Job Category Analysis
Shows which occupations generate the highest number of conversions.

### 4. Age Group Analysis
Evaluates conversion behavior across different age segments.

### 5. Education-wise Conversion Analysis
Compares conversion rates among various education levels.

### 6. Previous Campaign Outcome Analysis
Measures the impact of previous campaign outcomes on future conversions.

### 7. Marital Status Analysis
Provides insights into conversion performance across marital categories.

### 8. Interactive Filters
Users can filter dashboard results using:
- Month
- Job
- Education
- Contact Method
